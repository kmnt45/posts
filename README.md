# Posts App

Тестовое задание на `Next.js + React + TypeScript + CSS Modules`.

В проекте реализовано:

- загрузка постов с `https://jsonplaceholder.typicode.com/posts`
- вывод по 10 постов на страницу
- пагинация
- страница отдельного поста
- возврат на ранее выбранную страницу списка
- кэширование запросов средствами `fetch` в Next.js

## Запуск

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу `http://localhost:3000`.
